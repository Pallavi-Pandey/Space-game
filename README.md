# Pygame Space Shooter

An interactive, 2D space shooter game built using Python and the Pygame library.

## Project Structure

- `main.py`: The primary game loop, including player movement, enemy spawning, and collision detection.
- **Assets**:
  - `background.png` / `background.wav`: Atmospheric background visuals and music.
  - `bullet.png` / `laser.wav`: Projectile graphics and sound effects.
  - `enemy.png` / `ufo.png` / `explosion.wav`: Enemy sprites and destruction sounds.

## Key Features
- **Dynamic Gameplay**: Increasing difficulty and random enemy movement.
- **Audio Integration**: Full sound effect and background music support.
- **Visuals**: Animated sprites and smooth 2D rendering.

## Getting Started

1.  **Install Pygame**:
    ```bash
    pip install pygame
    ```
2.  **Run the Game**:
    ```bash
    python main.py
    ```
